# Weather Bot 🌤️

Telegram-бот для просмотра погоды через OpenWeatherMap API.

## Возможности

- 🌡️ **Текущая погода** — по названию города или геолокации
- 📅 **Прогноз на 5 дней** — с шагом 3 часа
- 🔄 **Сравнение городов** — сравнение температуры между двумя городами
- 🍃 **Расширенные данные** — загрязнение воздуха, влажность, давление, ветер
- 🔔 **Уведомления** — периодические уведомления о погоде (настраиваемый интервал)
- 💾 **Сохранение локации** — бот запоминает ваше местоположение

## Требования

- Python 3.10+
- Telegram Bot Token
- OpenWeatherMap API Key

## Установка

### 1. Клонируйте репозиторий

```bash
git clone https://github.com/username/weather-bot.git
cd weather-bot
```

### 2. Создайте виртуальное окружение

```bash
python3 -m venv .venv
source .venv/bin/activate  # macOS/Linux
# или
.venv\Scripts\activate  # Windows
```

### 3. Установите зависимости

```bash
pip install -r requirements.txt
```

### 4. Настройте переменные окружения

Скопируйте файл `.env.example` в `.env`:

```bash
cp .env.example .env
```

Заполните `.env` своими ключами:

```env
API_KEY=your_openweather_api_key
BOT_TOKEN=your_telegram_bot_token
```

### 5. Получите API-ключи

**Telegram Bot Token:**
1. Откройте [@BotFather](https://t.me/BotFather) в Telegram
2. Отправьте `/newbot`
3. Следуйте инструкциям
4. Скопируйте полученный токен

**OpenWeatherMap API Key:**
1. Зарегистрируйтесь на [openweathermap.org](https://openweathermap.org/)
2. Перейдите в раздел API Keys
3. Скопируйте ваш ключ

### 6. Запустите бота

```bash
python bot.py
```

## Использование

1. Откройте вашего бота в Telegram
2. Отправьте `/start`
3. Выберите нужную опцию из меню:

| Кнопка | Описание |
|--------|----------|
| Текущая погода | Показать погоду в указанном городе |
| Прогноз на 5 дней | Прогноз погоды на 5 дней |
| Моя геолокация | Отправить свою геопозицию и получить погоду |
| Сравнить города | Сравнить температуру в двух городах |
| Расширенные данные | Подробная информация (загрязнение воздуха, УФ-индекс) |
| Уведомления | Настроить периодические уведомления |

## Важные замечания

⚠️ **Безопасность:**
- Никогда не загружайте файл `.env` в публичный репозиторий
- Файл `user_data.json` содержит персональные данные пользователей
- При утечке API-ключей — немедленно перевыпустите их

⚠️ **API лимиты:**
- OpenWeatherMap Free тариф: 60 вызовов/минуту
- При превышении лимита бот автоматически делает повторные попытки
